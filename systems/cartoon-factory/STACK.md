# Cartoon Factory Stack

## Fixed Tools

### Images

Use ChatGPT Images 2.0 through the subscribed OpenClaw GPT image route for:

- canonical character sheets;
- facial expression sheets;
- pose sheets;
- background key art;
- prop sheets;
- scene keyframes when needed.

Do not use the legacy local image CLI path unless the operator explicitly changes this rule.

### Video

Use Gemini Video Generation Tool for scene clips.

Gemini should not invent character identity from scratch. Each scene prompt must reference:

- approved character card;
- approved reference image set;
- exact scene action;
- camera direction;
- continuity constraints;
- negative constraints.

### Voice And Sound Effects

Use ElevenLabs for:

- character voice lines;
- narrator voice, if used;
- sound effects where appropriate.

Each character must have a stable voice ID before production starts.

### Music

Use the approved music generation route for:

- intro/outro stingers;
- underscore;
- scene-specific mood beds.

Music must be mixed under dialogue and must not hide voice clarity.

## Output Assumption

Initial target format:

- duration: 45-60 seconds;
- aspect ratio: 9:16 by default for Shorts/Reels/TikTok;
- optional alternate format: 16:9 after the vertical version works;
- language: Ukrainian by default unless specified otherwise.
