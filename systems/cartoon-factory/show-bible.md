# Show Bible

## Purpose

This file defines the durable creative rules for the cartoon factory. Update it only when the whole show's direction changes.

## Default Format

- Episode length: 45-60 seconds for MVP.
- Default aspect ratio: 9:16.
- Language: Ukrainian.
- Structure: fast hook, clear conflict, escalation, twist, clean ending.
- Style target: visually consistent, readable, expressive, not overcomplicated.

## Story Rules

- Start with a concrete situation, not exposition.
- Give the main character a clear want in the first 10 seconds.
- Escalate through visible actions, not only dialogue.
- Include at least one visual joke, reversal, or callback.
- End with a clean final beat, not an abrupt stop.

## Character Continuity Rules

- Character silhouette, face shape, age, outfit, and core color palette must not change between scenes.
- Each character must have a canonical card before appearing in an episode.
- Scene prompts may change pose, emotion, lighting, and action, but not identity.
- If a generated clip changes identity noticeably, reject or regenerate it.

## Voice Rules

- Every speaking character needs a stable voice ID.
- Voice direction must include emotion, speed, pauses, and emphasis.
- Generate separate voice tracks per character when practical.
- Edit video timing around approved voice, not the other way around.

## Transition Rules

- Every scene must define its incoming transition and outgoing transition.
- Avoid random cuts that break spatial or emotional continuity.
- Prefer simple, readable transitions for MVP: hard cut, match cut, push-in, whip pan, sound bridge.

## QA Standard

An episode is not ready if:

- the same character looks like a different person;
- dialogue timing does not match scene length;
- music competes with speech;
- transitions feel accidental;
- captions cover faces or key action;
- the first 3 seconds are weak;
- the ending feels unfinished.
