# Agent Task: Platform Adapter

## Role
Adapt each approved content item to target platforms.

## Platform rules
- TikTok: fastest hook, raw demo energy, simple language.
- Instagram: cleaner packaging, trust + DM CTA.
- Facebook: more context and business value.
- Threads: short founder POV / opinion / conversation starter.

## Output
`platform_pack.json` with one object per platform:
- integration key from `orchestrator.yaml`
- caption/text
- media path
- scheduled_at suggestion
- post type
- CTA
