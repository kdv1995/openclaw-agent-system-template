# Runbook: Create Episode

## Goal

Create a production package for one short animated episode.

## Steps

1. Create a new folder under `episodes/` from `episode-template/`.
2. Fill `episode-brief.md`.
3. Draft `beat-sheet.md`.
4. Write `script.md`.
5. Create `timing-map.json`.
6. Confirm characters in `characters-used.md` or episode brief.
7. Generate or select required image references.
8. Write `image-prompts.md`.
9. Write `video-prompts.md`.
10. Write `voice-script.md`.
11. Write `sfx-map.md` and `music-brief.md`.
12. Create `edit-plan.md`.
13. Run continuity QA before generation.
14. Generate assets using the fixed stack.
15. Run final `qa-report.md`.

## Approval Rule

Do not publish externally without explicit approval from the operator.
