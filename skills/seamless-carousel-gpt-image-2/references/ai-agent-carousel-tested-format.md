# AI Agent Carousel Tested Format

Tested with the operator on 2026-05-23.

## Purpose

Professional Instagram-first carousel for the operator's personal AI/automation content. The goal is to test reaction quickly while keeping the visual quality high enough for Instagram, TikTok, Facebook, and Threads reposting.

## Structure

1. Slide 1: pain/hook.
2. Slide 2: mechanism/reveal.
3. Slide 3: payoff/CTA.

Use the "slippery slope" logic:

- Slide 1 creates tension and makes the problem obvious.
- Slide 2 explains the agent mechanism without over-teaching.
- Slide 3 gives one action, usually a comment/DM CTA.

## Visual System

- Premium SaaS/editorial carousel.
- Warm white/off-white background by default.
- Deep black typography.
- Vivid red accent only for rails, nodes, emphasis, or CTA.
- Abstract business UI cards, CRM blocks, lead indicators, queues, approvals, and pipeline elements.
- Red interface rail continues from slide to slide.
- No robots, no sci-fi cliches, no neon, no logos, no generated letters/numbers.

## Text Overlay

- Add all text locally after image generation.
- Use Unbounded 900 or closest Unbounded heavy weight. This is the approved carousel display font; do not use Outbound.
- Make text large enough to read from a phone feed.
- Prefer uppercase for this format.
- Keep text in the upper-left/center-left safe area.
- Avoid paragraphs inside the image.
- Match the approved reference: square/blocky Cyrillic, very heavy strokes, tight line-height, negative tracking, black headline with one decisive red phrase or word.

## Dark Format Variant

Use the same structure and typography, but switch the visual system:

- Deep black/charcoal background.
- Off-white UI cards and white typography.
- Red rail remains the main continuity device.
- One red CTA node or emphasis word on the final slide.
- Keep it premium and restrained; avoid cyberpunk, blue/purple gradients, and gaming aesthetics.
