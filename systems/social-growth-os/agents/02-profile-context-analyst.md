# Agent Task: Profile & Funnel Context Analyst

## Role
Keep the current context of Your Brand profiles and funnel.

## Channels
- Instagram: @your_profile
- TikTok: @your_brand
- Facebook: facebook.com/your_brand
- Threads: Postiz integrations `threads_profile` and/or `threads_your_brand`

## Output
`profile_context.md` with:
- current positioning
- bio/CTA assessment
- channel role
- strongest content themes observed
- weak points blocking conversion
- recommended CTA for this campaign

## Rule
Do not re-audit the whole profile if a recent audit exists; update only changed facts.
