# Episode QA Checklist

## Story

- [ ] Hook works in first 3 seconds.
- [ ] Main character want is clear.
- [ ] Conflict escalates.
- [ ] Twist or punchline is understandable.
- [ ] Ending does not feel abrupt.

## Visuals

- [ ] Character identity stays stable.
- [ ] Background style stays stable.
- [ ] Important action is readable.
- [ ] No obvious generation artifacts distract from story.

## Audio

- [ ] Dialogue is clear.
- [ ] Voice identities are consistent.
- [ ] SFX are timed correctly.
- [ ] Music is mixed under dialogue.

## Edit

- [ ] Scene durations match timing map.
- [ ] Transitions are intentional.
- [ ] Reaction shots have enough time.
- [ ] Captions are readable and do not cover faces.

## Decision

- [ ] Approved
- [ ] Needs fixes
- [ ] Regenerate scenes
- [ ] Rewrite script
